# Bebop Puzzle Maker

A visual puzzle piece maker for my upcoming game **Bebop Puzzle** — a music theory learning game based on 4-note jazz embellishment patterns.

## What is this?

This tool allows you to create and manage visual representations of 4-note melodic chunks commonly used in bebop jazz improvisation. Each piece combines:

- **Visual drawing** — Lines, dashed lines, and dots on a wood-grain canvas
- **Intervallic definition** — Define the musical intervals between each of the 4 notes
- **Labels** — Categorize patterns as Ascending, Descending, From Above, or From Below

## Features

- 🎨 **Drawing Tools** — Solid lines, dashed lines, and dots with customizable thickness and color
- 🎵 **Interval Selection** — Choose from all 12 intervals (Unison through Major 7th)
- 📚 **Library Management** — Save, load, and organize your puzzle pieces
- 💾 **Export Options** — PNG, SVG, or JSON (for the full library)
- 🔄 **Flip & Transform** — Horizontal and vertical flip with smooth animations
- ⌨️ **Keyboard Shortcuts** — Undo/Redo, Copy/Paste, Delete, and more

## Default Library

The app comes with a default library of common bebop patterns including:
- Neighbor tones
- Arpeggios
- Scale fragments
- Enclosures

## Usage

1. Open `index.html` in a browser
2. Use the drawing tools to create your visual pattern
3. Define the intervallic steps (1→2, 2→3, 3→4)
4. Give it a name and label
5. Click "Save to Library"

## Tech Stack

- Vanilla JavaScript
- Tailwind CSS
- HTML5 Canvas
- LocalStorage for persistence

## Future

This tool creates the puzzle pieces for **Bebop Puzzle**, a game where players will match melodic patterns to their visual representations, helping musicians internalize common jazz vocabulary.
